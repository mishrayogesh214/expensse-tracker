# Expense Tracker Web App

A web-based expense tracker built using FastAPI and Jinja2 templates.

## Features

- Add expenses
- Track spending
- View expense history
- FastAPI backend
- Simple UI

## Tech Stack

- Python
- FastAPI
- Jinja2
- HTML/CSS
- Pandas

## Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/expense-tracker.git

Install dependencies:

pip install -r requirements.txt

## Run

uvicorn main:app --reload

## Open in Browser

http://127.0.0.1:8000
